# LaslesVPN Landing Page

A responsive LaslesVPN landing-page exercise built with **Next.js**, React, and CSS. The implementation was produced collaboratively with an AI coding agent.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — start the local development server
- `npm run lint` — run ESLint
- `npm run build` — make a production build
